## Propósito do Código
O propósito deste projeto é atuar como um analista de dados para a loja online Ice, que vende videogames no mundo todo. O objetivo central é identificar padrões nas vendas de jogos em diferentes regiões para determinar estratégias de marketing viáveis.

O código analisa dados históricos de vendas de jogos, avaliações de usuários/críticos e classificações indicativas (ESRB) para:

Identificar quais plataformas de console estão em declínio e quais estão crescendo.

Determinar o impacto das avaliações feitas por usuários e por críticos profissionais nas vendas.

Traçar perfis de consumo regionais.

E com tudo isso, prever tendências para o ano de 2017, para permitir que a Ice planeja melhores campanhas publicitárias.

## Tecnologia usadas:
Python

Pandas: Processamento e manipulação de tabelas.

Matplotlib & Seaborn: Criação de gráficos de dispersão, boxplots e histogramas.

SciPy (stats): Execução de testes t para verificar igualdade de médias e validar hipóteses.
